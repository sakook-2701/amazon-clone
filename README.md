## Project Overview
This project is a frontend clone of the Amazon shopping website developed using HTML, CSS, and JavaScript. It replicates the basic layout and structure of an e-commerce platform.

## Features
- Responsive Navigation Bar
- Product Listing Section
- Banner and Category Sections
- Shopping Cart UI
- Footer Section
- Basic Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Objective
The objective of this project is to strengthen frontend development skills and understand the structure and design of real-world e-commerce websites.

## Project Structure
- index.html
- style.css
- script.js
- images/

## How to Run
1. Clone or download the repository.
2. Open index.html in any web browser.
